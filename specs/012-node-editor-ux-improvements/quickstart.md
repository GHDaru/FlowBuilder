# Quickstart: Node Editor UX Improvements

## 📋 Copying Content
1. Open the **Flow Builder**.
2. Select an **LLM Node** or **Condition Node** to open the configuration sidebar.
3. Locate the **Prompt Template** or **JSON Schema** field headers.
4. Click the **Copy** icon next to the header.
5. Notice the icon change to a checkmark briefly, confirming the content is in your clipboard.

## 📝 Expanded Editing
1. Open the configuration sidebar for a node.
2. Locate the **Prompt Template** or **JSON Schema** field headers.
3. Click the **Expand** icon next to the header.
4. A large dialog will appear containing the text.
5. Edit the text comfortably.
6. Click **Salvar** to apply the changes to the sidebar field, or press `Escape` to discard changes and close the dialog.
