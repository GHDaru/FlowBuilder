# Quickstart: LLM Node Temperature Control

## 🌡️ Adjusting Temperature
1. Open the **AI Visual Lab**.
2. Create or select an **LLM Node**.
3. In the right sidebar, locate the **Temperatura** section.
4. Move the slider to the right to increase creativity (e.g., `0.7` for summaries) or keep it at `0.0` for maximum consistency (e.g., data extraction).
5. Click **Salvar Configuração**.

## 🧪 Verifying Execution
1. Run a flow containing the configured node.
2. (Technical) Verify the backend logs or provider-specific metrics to confirm that the temperature value was passed to the AI model.
