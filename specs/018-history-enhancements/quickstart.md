# Quickstart: Execution History Enhancements

## 🕵️ Auditing with the New Header
1. Navigate to the **Histórico** tab.
2. Select any recent execution.
3. Observe the new header section. You should see:
    - **Contabilidade**, **Cliente**, and **Atendente**.
    - The **Flow Name** used for the evaluation.
    - The **Total Execution Time** (e.g., `8.2s`).

## ⏱️ Checking Per-Node Performance
1. Expand any step (Accordion) in the execution trace.
2. In the step header, alongside the token counts, look for the **Duração** label (e.g., `1.4s`).

## 📦 Exporting all JSONs
1. Locate the **"Copiar Tudo (JSON)"** button in the history detail header.
2. Click the button.
3. Paste the content into any text editor. Verify it is a consolidated JSON containing the outputs of every node in the trace.
