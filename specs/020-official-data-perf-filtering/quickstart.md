# Quickstart: Official Data Performance & Filtering

## ⚡ Fast Searching for Firms
1. Open the **Base Oficial** tab.
2. Type part of an accounting firm name in the search box.
3. Observe the loading spinner appearing briefly.
4. The list updates instantly with only matching firms, regardless of how many thousands of records exist in the database.

## 🔍 Filtering by Status
1. Select one or more firms and click **Ver Atendimentos**.
2. Above the interaction table, locate the **Filtrar Status** dropdown.
3. Change to **Pendentes** to see only interactions without an evaluation.
4. Change to **Avaliados** to see only those already processed.
5. Use the header checkbox to select all *visible* items only.
