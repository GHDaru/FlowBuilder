# Quickstart: AvaliacaoBlock Component

## 🧪 Testing with Mock Data
1. Start the **AI Visual Lab** frontend and backend.
2. Navigate to the **Histórico** tab.
3. Select an execution that resulted in a consolidation JSON.
4. The `AvaliacaoBlock` should automatically render below the interaction title.

## 🌳 Interacting with the Audit Tree
1. Locate the **Árvore de Auditoria** section.
2. Click on **Expandir tudo** (top right) to see the full hierarchy.
3. Click on **Recolher tudo** to collapse back to the root.
4. Manually click on a **Dimensão** to see its phrase summary.
5. Click on a **Sub-eixo** to see the technical justification and any rules (highlighted in colors like blue, orange, or red).

## 🎨 Visual Verification
- Verify that **exemplar** (10.0) shows a green badge.
- Verify that a **promotor** classification shows a green indicator.
- Verify that **alerta_imediato** shows a prominent red highlight if active.
