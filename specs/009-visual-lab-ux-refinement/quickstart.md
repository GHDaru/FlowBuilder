# Quickstart: Visual Lab UX/UI Refinement

## 🎨 Using the New Editor Features

1. **Contextual Toolbar**:
   - Open any flow.
   - Click on an LLM Node.
   - A toolbar will appear above the node with Duplicate and Delete options.

2. **Inline Renaming**:
   - Look at the top left of the editor header.
   - Click the flow name or the pencil icon.
   - Type a new name and press **Enter**.

3. **Code Export**:
   - Click the `</>` icon in the top right header.
   - Use the **Copiar** button to put the JSON in your clipboard.
   - Use the **Baixar .json** button to save a local backup.

4. **Batch Operations**:
   - In the **Meus Fluxos** screen, use the **Importar Fluxo** button to upload a JSON file.
   - Use the **Exportar** icon on any flow card to download its configuration.

## ⌨️ Keyboard Shortcuts
- `Ctrl + S`: Save the current flow.
- `Ctrl + Shift + S`: Save flow as (opens dialog).
